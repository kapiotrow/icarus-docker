### Running sims
To run the prepared simple sim, reopen the repo in container, open a new terminal and type 'make' and then 'make wave'. The GTKWave GUI should open up. Find the module's name in the top left corner, double click it. Navigate to the bottom left corner and double click on all of the signals you want to see in the wave inspector.
